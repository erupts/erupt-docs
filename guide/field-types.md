# 字段类型总览

> 占位文档，内容待补充。

| 类型 | 说明 |
|------|------|
| `INPUT` | 单行文本输入 |
| `TEXTAREA` | 多行文本 |
| `NUMBER` | 数字输入 |
| `SLIDER` | 滑块 |
| `DATE` | 日期选择 |
| `DATE_RANGE` | 日期范围 |
| `BOOLEAN` | 布尔值 |
| `SWITCH` | 开关 |
| `CHOICE` | 单选（下拉/Radio）|
| `TAGS` | 多选标签 |
| `ATTACHMENT` | 附件上传 |
| `IMAGE` | 图片上传 |
| `REFERENCE_TABLE` | 关联表选择 |
| `CODE_EDITOR` | 代码编辑器 |
| `HTML_EDITOR` | 富文本编辑器 |
| `MAP` | 地图坐标 |
