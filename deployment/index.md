# 部署概览

Erupt 框架支持多种部署方式，可根据业务规模和团队技术栈灵活选择。

## 部署方式

| 部署类型 | 说明 |
| --- | --- |
| 单体部署 | 默认方式，前后端一体，适合中小规模项目 |
| 前后端分离 | 前端独立部署，适合有定制化前端需求的场景 |
| 分布式部署 | 通过 erupt-cloud 实现多节点集群管理 |

## 数据库支持

erupt 支持市面上所有主流数据库，详见：[数据源支持](/deployment/database)
