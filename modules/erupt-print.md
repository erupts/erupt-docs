# Erupt Print 打印模块

:::info
erupt-print 目前作为 [erupt-flow](/modules/pro/erupt-flow) 的打印能力支撑模块，为流程单据提供模板化打印功能。
:::

erupt-print 提供基于 HTML 模板的打印能力，支持配置模板变量，将业务数据动态填充后生成可打印的单据或报告。
