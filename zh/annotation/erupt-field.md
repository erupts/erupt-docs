# @EruptField

字段级注解，用于配置字段在表格中的展示（`@View`）和表单中的编辑行为（`@Edit`）。

## 属性说明

| 属性名 | 描述 |
|--------|------|
| `views` | 表格列配置，格式为数组，支持配置多个列；不配置则不在表格中展示 |
| `edit` | 编辑组件配置；不配置则不展示表单组件，新增/编辑时框架不信任任何前端传值 |
| `sort` | 展示顺序，数字越小越靠前，默认按字段声明顺序排列 |
| `params` | 自定义扩展参数，会透传给前端 |

## 示例

```java
@EruptField(
    sort = 10,
    views = @View(title = "名称"),
    edit = @Edit(title = "名称", notNull = true)
)
private String name;
```

## 注意事项

- `views` 去掉则字段不在表格中展示
- `edit` 去掉则字段不可在表单中编辑，且框架不会信任前端提交的该字段值
