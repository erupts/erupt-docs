# 进阶

本章介绍 Erupt 的进阶用法与扩展开发，帮助你在实际项目中应对更复杂的业务场景。

## 业务扩展

| 功能 | 说明 |
|------|------|
| [DataProxy 数据代理](/advanced/data-proxy) | 在增删改查各阶段注入自定义业务逻辑 |
| [动态表单 OnChange](/advanced/dynamic-form) | 根据条件动态控制字段显示、校验、选项等 |
| [自定义文件上传](/advanced/upload) | 对接 OSS、本地存储等自定义上传实现 |
| [事件监听器](/advanced/event-listener) | 监听框架内部事件，实现解耦扩展 |
| [国际化（i18n）](/advanced/i18n) | 多语言支持与自定义翻译 |
| [杂项功能](/advanced/misc) | 其他实用功能汇总 |

## 集成接入

| 功能 | 说明 |
|------|------|
| [现有项目接入](/advanced/integration) | 将 Erupt 集成进已有 Spring Boot 项目 |
| [登录与认证](/advanced/auth) | 自定义登录逻辑、Token 校验、权限扩展 |
| [多数据源](/advanced/datasource) | 配置并使用多个数据库数据源 |
