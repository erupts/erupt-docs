# 简介

Erupt 是一款基于注解驱动的低代码 Java 框架，通过少量注解配置即可自动生成企业级后台管理系统。

## 核心理念

> 写注解，不写代码。

传统后台开发需要编写大量重复的 CRUD 代码，Erupt 将这些工作抽象为注解配置，开发者只需关注业务字段的定义，框架负责自动生成所有接口和前端页面。

## 功能特性

- 零前端代码，自动渲染管理页面
- 注解驱动，极简 API
- 内置用户权限体系（UPMS）
- 支持多种数据库
- 丰富的字段编辑器类型
- 强大的扩展机制

## 适用场景

适合企业内部管理系统、运营后台、数据管理平台等快速开发场景。
